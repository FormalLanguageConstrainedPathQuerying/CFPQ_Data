cfpq_data.grammars.utils.change_terminals.change_terminals_in_cfg(cfg: pyformlang.cfg.cfg.CFG, spec: Dict[str, str]) → pyformlang.cfg.cfg.CFG[source]¶

Change terminals of a context-free grammar [1].

Parameters

  • cfg (CFG) -- Context-free grammar.

  • spec (Dict) -- Terminals mapping.

Examples

>>> import cfpq_data
>>> cfg = cfpq_data.cfg_from_text("S -> a S b S")
>>> new_cfg = cfpq_data.change_terminals_in_cfg(cfg, {"a": "b", "b": "c"})
>>> new_cfg.to_text()
'S -> b S c S\n'
